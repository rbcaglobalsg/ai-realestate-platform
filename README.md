| **누적 투자** | 시드 단계 — 골든 타임 | **₩102억 (2020년 Series A 이후 0원)** |
| **현재 상태** | 🟢 활발 개발 중 | 🔴 핵심 서비스 리뉴얼 중단, 인재 유출 |

| **AI 기술** | 멀티모달 AI (수익성 분석 + 설계 생성 + 시장 예측) | Deep RL 설계 최적화 (단일 기능) |
| **데이터 소스** | 4대 공공 API (실거래가·건축물대장·공시지가·토지이용계획) + AI 분석 | 공공 데이터 + 자체 알고리즘 |
| **수익 모델** | Freemium SaaS + 프리미엄 AI 분석 + B2B API | B2G 중심 (37개 계약) |
| **시장 확장성** | 개인 투자자 → 중개사 → 개발사 → 금융권 → 공공 | 소규모 토지 개발 B2B/B2G |
| **국제 확장** | 아키텍처 자체가 다국가 확장 가능 | 한국 전용 (확장 증거 없음) |
| **특허** | 개발 중 | 16건 등록 |
| **투자 타이밍** | **지금이 골든 타임** — 경쟁사 약체화 + 시장 반등 | **6년 무투자** — 투자자 신뢰 하락 |

### 왜 지금인가?

1. **SpaceWalk 리뉴얼 공백** — 랜드북 서비스 중단 상태에서 사용자 이탈 가속화
2. **한국 부동산 시장 반등** — 2025년 거래금액 ₩448.4조 (전년 대비 +18.8%)
3. **프롭테크 투자 회복** — 2023년 ₩3,090억 바닥 → 2024년 ₩8,000억 회복
4. **AI 프롭테크 빈집** — 한국에는 아직 Zestimate/Compass AI 급의 서비스가 없음
5. **공공데이터 무료** — 실거래가·건축물대장·VWorld·토지이용계획 전부 무료 API

### AI 기술 스택

- **수익성 분석**: OpenAI GPT-4.1 + Claude Opus — 공공데이터 기반 실시간 수익성 계산
- **설계 생성**: FAL AI (Flux Pro) — 텍스트→건축 설계 이미지, 3D 렌더링
- **시장 예측**: ML 시계열 분석 — 2006년~현재 실거래가 데이터 기반 예측 모델
- **자연어 인터페이스**: "강남구 30평 대지에 빌라 지으면 수익 얼마나 나?" → 즉시 분석

## 🇰🇷 한국 공공 API — 전부 무료

한국은 세계 최고 수준의 부동산 공공데이터 인프라를 보유하고 있습니다. **전부 무료**입니다.

| API | 제공기관 | 데이터 | 업데이트 |
|-----|---------|--------|---------|
| **실거래가** | 국토교통부 | 아파트·연립·단독·오피스텔·토지·상업 매매/전월세 | 월 1회 |
| **건축물대장** | 국토교통부 | 건물 개요·연면적·용도·구조·승인일 | 월 1회 |
| **VWorld** | 국토교통부 | 공간정보 28종 — 지적·가격·토지형상·건물융합 | 실시간 |
| **토지이용계획** | 국토교통부 | 용도지역·개발제한·건폐율·용적률 | 반기 1회 |
| **개별공시지가** | 국토교통부 | 필지별 공시지가 (~4,000만 필지) | 반기 1회 |
| **법정동코드** | 행정안전부 | 전국 법정동 코드 | 월 1회 |
| **LH 분양정보** | 한국토지주택공사 | 분양임대공고·사전청약 | 실시간 |

> **경쟁 우위**: 미국 Zillow는 데이터 확보에 연간 수백만 달러를 지출하지만, 한국은 **정부가 무료로 제공**합니다. 이 데이터를 AI로 가공하는 것이 우리의 핵심 가치 창출입니다.

### 기술 스택 상세

**프론트엔드**
- React 19 + Vite 7 + TailwindCSS 4
- React Router 7 (SPA + SSR 준비)
- Recharts / D3.js (데이터 시각화)
- Mapbox / Leaflet (지도 연동)

**백엔드**
- Express 5 + TypeScript + Drizzle ORM
- PostgreSQL (Railway 관리형)
- Redis (API 응답 캐시)
- JWT 인증 + 구독 관리

**AI / ML**
- OpenAI GPT-4.1 / Claude Opus (수익성 분석, 자연어 인터페이스)
- FAL AI Flux Pro (설계 이미지 생성)
- Custom ML Pipeline (시계열 가격 예측)
- Embedding + Vector Search (유사 사례 검색)

**인프라**
- Railway (백엔드 자동 스케일링)
- Netlify / Vercel (프론트엔드 CDN)
- Toss Payments (결제)
- GitHub Actions (CI/CD)

- Node.js 20+

- 공공데이터포털 API Key ([data.go.kr](https://data.go.kr)에서 무료 발급)

### 설치

```bash
# 클론
git clone https://github.com/rbcaglobalsg/ai-realestate-platform.git
cd ai-realestate-platform

# 백엔드 설정
cd backend
cp .env.example .env
# .env에 FAL_API_KEY, 공공데이터 API 키 입력
npm install
npm run dev

# 프론트엔드 설정 (새 터미널)
cd frontend
npm install
npm run dev
```
